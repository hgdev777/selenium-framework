selenium-framework
==================

Selenium wrapper to make selenium coding simple and easy


Dependencies
============

1. JDK 7 or greater

2. Maven 3.0.5 or greater


Instructions
============

Run all suites
```sh
mvn test -DsuiteXmlFile=configs/suites/suites.xml
```


Run a suite
```sh
mvn test -DsuiteXmlFile=configs/suites/suiteGoogle.xml
```
